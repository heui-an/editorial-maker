# 🌍 상세페이지 메이커 (클레이 지구 만들기 템플릿)

Figma 「상세페이지-1240」 디자인을 그대로 재현한 **템플릿형 상세페이지 제작 도구**입니다.
왼쪽에서 **텍스트와 이미지만 바꾸면** 오른쪽 미리보기에 바로 반영되고,
**버튼 한 번으로 전체를 하나의 PNG 이미지(1240px 폭, 원본 1:1)** 로 저장합니다.

---

## ▶ 실행 방법

### 가장 쉬운 방법 (Mac)
1. `실행하기.command` 파일을 **더블클릭**합니다.
2. 자동으로 브라우저가 열립니다. (안 열리면 주소창에 `http://localhost:8765` 입력)
3. 다 쓰면 검은 터미널 창을 닫으면 종료됩니다.

> 처음 더블클릭 시 “확인되지 않은 개발자” 경고가 뜨면
> `시스템 설정 → 개인정보 보호 및 보안`에서 **“확인 없이 열기”** 를 눌러주세요.

### 직접 실행 (터미널)
```bash
cd detail-maker
python3 -m http.server 8765
# 브라우저에서 http://localhost:8765 접속
```

> ⚠️ `index.html`을 그냥 더블클릭(file://)하면 **이미지 저장이 동작하지 않습니다.**
> 반드시 위 방법(로컬 서버)으로 열어주세요. 인터넷 연결이 필요합니다(본문 폰트 로드).

## ✏️ 사용법

| 기능 | 설명 |
|------|------|
| **텍스트 수정** | 왼쪽 입력칸에 입력하면 즉시 반영됩니다. Enter(줄바꿈)도 그대로 반영돼요. |
| **강조색** | 글자를 `*별표*`로 감싸면 그 부분만 강조색(섹션 포인트 컬러)으로 표시됩니다.<br>예) `지구는 *양파처럼* 생겼어요` → ‘양파처럼’이 강조색 |
| **이미지 변경** | 각 이미지 항목의 **[이미지 변경]** 버튼으로 내 PC 이미지를 올립니다. **[기본값]** 으로 되돌릴 수 있어요. |
| **🖼️ 이미지로 저장 (PNG)** | 전체 페이지를 하나의 PNG 파일로 다운로드합니다. (`상세페이지_날짜.png`) |
| **설정 저장 / 불러오기** | 작업 내용을 JSON 파일로 저장해두고 나중에 그대로 불러올 수 있습니다. |
| **전체 초기화** | 모든 내용을 원본 디자인으로 되돌립니다. |

> 작업 내용은 브라우저에 자동 저장되어, 창을 닫았다 열어도 유지됩니다.

## 🔤 폰트 안내 (선택)

원본 디자인은 **산돌 / Paperlogy** 유료 폰트를 사용합니다.
이 폰트가 없으면 비슷한 **무료 대체 폰트(Jua·Gaegu·Pretendard)** 로 표시됩니다 (모양은 거의 동일).

**원본과 100% 동일하게** 하려면, 보유한 폰트 파일을 `fonts/` 폴더에 아래 이름으로 넣어주세요
(`.woff2` 또는 `.otf`/`.ttf` 모두 가능):

| 파일명 | 용도 |
|--------|------|
| `SandollCinemaTheater` | 큰 타이틀 · 바 제목 |
| `SandollGyeokdongGL2` | 강조 헤드라인 · 카드 제목 |
| `SandollGyeokdonggulim` | POINT 라벨 |
| `SandollNemony2` | 구조 라벨(지각/맨틀…) |
| `SandollBaikzongyulPen` | 손글씨 말풍선 |
| `Paperlogy-7Bold` | 교과연계 원형 뱃지 |

넣은 뒤 브라우저를 새로고침하면 자동 적용됩니다. (자세한 매핑은 `css/fonts.css` 참고)

## 📁 폴더 구조

```
detail-maker/
├─ 실행하기.command     ← 더블클릭 실행 (Mac)
├─ index.html           ← 메인 페이지
├─ css/
│  ├─ page.css          ← 상세페이지 디자인 (Figma 1:1 재현)
│  ├─ app.css           ← 편집기 UI
│  └─ fonts.css         ← 폰트 설정 (산돌/Paperlogy 슬롯)
├─ js/
│  ├─ data.js           ← 기본 콘텐츠 + 편집 항목 정의  ← 항목 추가/수정은 여기
│  ├─ render.js         ← 콘텐츠 → 페이지 생성 로직
│  └─ app.js            ← 편집/저장/PNG 내보내기 로직
├─ assets/              ← 디자인 이미지(배경·아이콘·일러스트·제품 컷아웃)
├─ fonts/               ← (선택) 산돌/Paperlogy 폰트 파일을 여기에
└─ vendor/
   └─ html-to-image.js  ← PNG 변환 라이브러리
```

## 🛠 자주 묻는 질문

- **글자가 잘리거나 줄이 이상해요** → 입력칸에서 직접 Enter로 줄바꿈 위치를 맞춰주세요.
- **이미지가 칸에 꽉 안 차요** → 이미지는 칸을 꽉 채우도록(cover) 들어갑니다. 비율이 비슷한 사진을 권장합니다.
- **저장이 안 돼요** → `file://`로 열지 않았는지 확인하세요. 반드시 `실행하기.command` 또는 로컬 서버로 여세요.
- **다른 상품용으로 쓰고 싶어요** → 텍스트/이미지를 모두 바꾼 뒤 **설정 저장**으로 보관하면, 상품별로 관리할 수 있습니다.
